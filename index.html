Partes de um computador (clique para expandir)

Contrair todos
  • Placa-Mãe (Motherboard)
  • Processador (Central Processing Unit: CPU)
  • Memória (Random Access Memory: RAM)
  • Alojamento (Solid State Drive: SSD)
  • Placa de Vídeo (Graphical Processing Unit: GPU)
    Nota : GPUs estão bem caras recentemente devido à escassez dessas no mercado, logo os preços estão bastantes inflados mesmo para padrões brasileiros.
    Um Computador não necessariamente necessita de uma GPU para funcionar, contudo, sem uma GPU a inteiridade de processamento gráfico (ex: Renderização, Streaming, Gameplay, ...) fica para a CPU.
  • Suprimento (Power Supply)
  • Ventilação e Pasta Térmica (Cooler & Thermal Paste)
    Nota : O preço é total (Cooler + Pasta).
  • Gabinete (Case)
  • Computadores pré-montados (Builds)
